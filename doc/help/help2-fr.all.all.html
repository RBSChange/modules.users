Comment créer un nouvel utilisateur de la partie protégée du site

Pour créer un nouvel utilisateur au sein du groupe d’utilisateurs enregistrés sur le site :

  • Sélectionner le groupe d’utilisateurs dans lequel on souhaite l’insérer
  • Cliquer sur dans la barre de fonctionnalités situé en haut à gauche de l’applicatif.

Pour le groupe d’utilisateurs au(x) site(s), la boîte des propriétés s’affiche :

  • Renseigner les champs obligatoires signalés par . Les champs non-valides sont marqués par l’icône.
  • Remplir les champs relatifs aux propriétés de l’utilisateur :
    • « Civilité » : Liste déroulante.
    • « Prénom » : Champ textuel libre.
    • « Nom » : Champ textuel libre.
    • « Identifiant (Adresse e-mail) » : Champ textuel libre se remplissant automatiquement par rapport aux champs prédéfinis en amont.
    • « Mot de passe » : Champ textuel libre pouvant se définir aléatoirement.
    • « Début de validité du compte » : Champ date. Cliquer sur pour afficher le calendrier électronique, sélectionner la date désirée pour son inscription automatique dans le champ date. Tant que ce champ n’est pas rempli, l’utilisateur ne pourra avoir accès à la partie protégée du site.
    • « Fin de validité du compte » : Champ date. Cliquer sur pour afficher le calendrier électronique, sélectionner la date désirée pour son inscription automatique dans le champ date. Si ce champ n’est pas rempli, l’utilisateur aura accès à la partie protégée du site d’une manière indéfinie sur le site, à partir du moment où son statut aura été activé.
  • Une fois l’ensemble des propriétés de l’utilisateur indiquées, possibilité est offerte de cliquer sur :
    • : Pour enregistrer les données
    • : Pour vider le contenu des champs et recommencer la saisie
    • : Pour enregistrer les propriétés de l’utilisateur tout en fermant ensuite la fenêtre des propriétés

Pour activer les statuts des utilisateurs :

  • Sélectionner un utilisateur en cliquant sur son nom au centre de l’applicatif
  • Cliquer sur

Pour désactiver un utilisateur, cliquer sur .

Pour le réactiver, cliquer sur .

Pour supprimer un utilisateur :

  • Cliquer sur son titre au centre de l’applicatif pour le sélectionner
  • Cliquer sur

Il est possible d’ordonner ensuite les différents utilisateurs insérés :

  • Sélectionner le répertoire dans lequel les utilisateurs ont été intégrés (sur la gauche de l’applicatif),
  • Cliquer sur
  • Utiliser les flèches afin de faire remonter ou de faire redescendre un utilisateur dans la liste
  • Cliquez sur pour valider
