Les groupes d’utilisateurs

Par défaut sur Change, deux groupes d’utilisateurs sont effectifs : le groupe d’utilisateurs de Change et le groupe d’utilisateurs enregistrés sur le site (grâce à une page faisant afficher un formulaire pour l’insertion d’un identifiant et d’un mot de passe).

Pour supprimer un groupe d’utilisateurs :

  • Cliquer sur son titre au centre de l’applicatif pour le sélectionner
  • Cliquer sur

Il est possible d’ordonner ensuite les différents groupes insérés :

  • Sélectionner le répertoire dans lequel ils ont été intégrés sur la gauche de l’applicatif
  • Cliquer sur
  • Utiliser les flèches afin de faire remonter ou de faire redescendre un groupe dans la liste
  • Cliquer sur afin de valider

Pour renommer un groupe d’utilisateurs, cliquer sur .
