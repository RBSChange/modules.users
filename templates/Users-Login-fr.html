Accéder à l'interface d'adminstration

Bienvenue dans RBS Change ! Pour accéder à l'interface d'administration de votre CMS, vous devez utiliser le navigateur Web Firefox. Firefox est le plus rapide, le plus sécurisé et le meilleur moyen de naviguer sur le Web. Vous pourrez alors installer l'extension RBS Change Manager qui vous permettra d'accéder au backoffice. Si vous souhaitez plus d'informations sur RBS Change Manager, vous pouvez consulter la documentation en ligne de l'extension.
